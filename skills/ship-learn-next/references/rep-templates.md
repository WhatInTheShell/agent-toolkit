# Rep Templates & Output Structure

## Rep Plan Template

```markdown
## Rep [N]: [Specific, Shippable Goal]

**Ship Goal**: [Concrete deliverable — a URL, file, sent message, published post]
**Timeline**: By [specific date, not "this week"]
**Success Criteria**:
- [ ] [Observable artifact exists]
- [ ] [Delivery action taken — published/sent/shared]
- [ ] [One measurement recorded]

**What You'll Practice** (from the content):
- [Skill/concept 1 mapped to source material]
- [Skill/concept 2 mapped to source material]

**Action Steps**:
1. [First concrete action — verb + noun + time estimate]
2. [Second concrete action]
3. [Third concrete action]
4. SHIP: [Specific delivery action — deploy/send/publish/share]

**Minimal Resources** (only what's needed for THIS rep):
- [Reference only if genuinely blocking — otherwise omit]

**After Shipping — Reflection**:
- What actually happened? (facts, not feelings)
- What worked? What didn't?
- What surprised you?
- Rep score: _/10
- One thing to try differently in Rep 2:
```

## Rep Progression Structure

```markdown
## Rep 2: [Next Iteration]
**Builds on**: [Specific insight from Rep 1 reflection]
**New element**: [ONE new variable introduced]
**Ship goal**: [Next concrete deliverable]

## Rep 3–5: Future Path
*(Outline only — details evolve from actual Rep 1-2 learnings)*
**Rep 3**: [Brief description]
**Rep 4**: [Brief description]
**Rep 5**: [Brief description]
```

## Quest Overview Template

```markdown
# Ship-Learn-Next Quest: [3-6 word title]

**Goal**: [What you want to achieve — measurable, 4-8 weeks]
**Source**: [Content that generated this plan]
**Core Lessons** (actionable only):
1. [Lesson → How to practice it]
2. [Lesson → How to practice it]
3. [Lesson → How to practice it]
```

## File Naming Convention

`Ship-Learn-Next Plan - [Brief Quest Title].md`

Examples:
- `Ship-Learn-Next Plan - Cold Email 10 Responses.md`
- `Ship-Learn-Next Plan - React Portfolio Launch.md`
- `Ship-Learn-Next Plan - Proven Market Validation.md`

Quest title: 3-6 words, action-oriented, outcome-focused (not topic-focused).

## Rep Sizing Reference

| Domain | Rep 1 Target | Typical Duration |
|--------|-------------|-----------------|
| Writing/content | 1 published piece | 2-3 days |
| Cold outreach | 5-10 sent messages | 1-2 days |
| Code/software | Deployed hello-world version | 3-5 days |
| Sales/bizdev | 1 real conversation | 1-3 days |
| Creative (design, audio) | 1 finished artifact shared | 2-4 days |
| Teaching/coaching | 1 session delivered | 1 week |

**Calibration rule**: If Rep 1 takes longer than 7 days, it's too big. Cut scope, not quality.
